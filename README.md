# blender-addons

A collection of Blender addons for various workflows and utilities.

## Available Addons

### 3DPrintToolKits

A comprehensive addon providing tools and utilities for 3D printing workflows.

**Features:**
- Sidebar integration in Blender's 3D View (press `N` key)
- Custom "3DPrintToolKits" tab for easy access
- Mesh analysis tools (placeholder buttons for thickness and normals checking)
- Export settings section (ready for future functionality)

**Installation:**
1. Open Blender
2. Go to `Edit` > `Preferences` > `Add-ons`
3. Click `Install...` and select the `__init__.py` file from the `3DPrintToolKits` folder
4. Enable the addon by checking the checkbox

**Requirements:** Blender 3.0 or higher

For more details, see [3DPrintToolKits/README.md](3DPrintToolKits/README.md)
